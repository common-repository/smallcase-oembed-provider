The plugin extends Wordpress's automatic embed feature, allowing you to directly embed smallcase cards from the embed URL

== Description ==

With smallcase's embed cards, you can display smallcase information & enable transactions directly on your WordPress site. It's as simple as copy-pasting a URL or code snippet!

By installing this plugin, you can easily embed cards by pasting the embed URL.

**Usage**

1. From the smallcase page, select embed icon from share panel
2. Customize the card as per your requirement
3. Select 'Copy URL' button
4. Paste the URL in your WordPress post editor ðŸŽ‰
